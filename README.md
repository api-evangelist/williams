# Williams Companies

Williams Companies is a Fortune 500 energy infrastructure company that operates approximately 33,000 miles of natural gas pipeline infrastructure, moving roughly one-third of the nation's natural gas. The company's flagship Transco pipeline system spans 10,000 miles from south Texas to New York City and is the nation's largest-volume natural gas pipeline. Williams provides pipeline transportation, gathering, processing, and storage services across key natural gas supply basins and serves utilities, power generators, industrial customers, and local distribution companies across the United States.

## APIs

### Transco Nominations API
The Transco Nominations API enables natural gas shippers to submit, modify, and track pipeline transportation nominations on the Transco pipeline system via the 1Line electronic bulletin board portal. Submissions are supported via online screen entry, flat file uploads, and EDI.

- **Human URL:** https://www.1line.williams.com/Transco/index.html

### Northwest Pipeline EBB
Northwest Pipeline provides electronic bulletin board access for nominations and capacity management on the Northwest Pipeline system serving the Pacific Northwest.

- **Human URL:** https://www.1line.williams.com/NWP/index.html
